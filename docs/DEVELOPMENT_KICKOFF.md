# 開発キックオフ（AI に渡す「最初の指示文」）

`docs/FIRST_AGENT_PROMPT.md` で **名前・デプロイの初期セットアップを済ませたあと**、機能開発に入るときに使います。  
**下の枠をそのままコピー**して AI に送ってください（加工不要で開始できます）。

---

## コピー用（この枠の中をすべてコピー）

```
あなたはこのリポジトリの実装担当です。次のとおり進めてください。

【読む順（着手前にすべて読む）】
1. 00_START_HERE.md
2. AGENTS.md
3. docs/開発依頼書.md（プロダクト仕様の正本）
4. docs/00_constitution.md
5. docs/01_history.md
6. docs/02_operations.md
7. docs/DESIGN_HANDOVER.md

【やること】
- docs/開発依頼書.md の「AI搭載開発ワークスペース」を実装対象とする。仕様と矛盾する実装にしない。
- docs/00_constitution.md を開発依頼書と矛盾しないよう、必要なら更新する。
- Phase 1 のゴール案: 依頼書の3核心を守ったうえで、トップ画面に「プロジェクト一覧カード」（ダミーデータでよい）を表示し、スマホで見やすいレイアウトにする。まずこの Phase 1 のゴールを1文で確認してから実装に入る。
- 本番の AI API・永続化 DB は Phase 1 ではモックまたはローカルでよい（依頼書のフェーズ方針に従う）。
- 作業完了時に docs/01_history.md に今日の日付でエントリを追記する。

憲法・履歴・デザイン憲法に従って実装し、終わったら docs/01_history.md に追記して。
```

---

## 任意（追記したいときだけ）

同じチャットの **先頭か末尾** に、次のような **1 文だけ**足してもよいです。

- 例: 「Phase 1 はカード3枚のダミーで十分」
- 例: 「まずは PC 幅だけでもよい」
