# my-workspace

**AI搭載開発ワークスペース**を実装するリポジトリです。プロダクト仕様の正本は **`docs/開発依頼書.md`** です。

## 最初に読む

1. **`00_START_HERE.md`**
2. 初回セットアップ（名前・デプロイ）: **`docs/FIRST_AGENT_PROMPT.md`**
3. **開発を始める**: **`docs/DEVELOPMENT_KICKOFF.md`** の指示文をコピーして AI に送る

## ローカル開発

```bash
npm install
npm run dev
```

## 技術スタック

Vite 6 + React 19 + TypeScript + Tailwind 3 + lucide-react + framer-motion。UI は **`docs/DESIGN_HANDOVER.md`** に従う。
